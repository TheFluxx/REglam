# Single-Developer UI/UX Review Sandbox

Локальный статический проект для одиночной фронтенд-разработки с формализованным разбором UI/UX ошибок.

## Запуск

1. Вариант без сервера: открыть `index.html` в браузере.
2. Рекомендуемый вариант:
```powershell
cd d:\REglam
python -m http.server 8080
```
Открыть: `http://localhost:8080`

## Что проверять

1. Наличие 4 секций: Hero, контент, форма/интерактив, футер.
2. 2 пользовательских сценария:
- быстрый путь через кнопку Hero к форме;
- альтернативный путь через меню и секцию возможностей.
3. Адаптивность на 360px и 1280px+.
4. Базовая доступность: заголовки, фокус, контраст, `alt`.
5. Логи дефектов и решений в `docs/03_review_log.md` и `docs/04_decisions.md`.

## Ограничения

1. Без backend, БД, хостинга.
2. Стек: `HTML/CSS/JS`.
3. Итог приемки: `Pass/Fail`.
