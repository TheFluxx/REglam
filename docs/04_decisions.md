# 04 Decisions

## Формат записей
- `Decision ID`
- Дата
- Контекст
- Варианты
- Выбор
- Обоснование
- Влияние
- Связанные `Issue ID` / `GAP ID`

- Контекст: Базовая архитектура проекта.
- Варианты: `Framework`, `HTML/CSS/JS`.
- Выбор: `HTML/CSS/JS`.
- Обоснование: Максимальная прозрачность и воспроизводимость для локального ревью.
- Влияние: Быстрый старт, меньше инфраструктуры, проще аудит.
